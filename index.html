Pequeno guia de front-end para alfaiataria digital.

Uma direção seguindo diversas fontes confiáveis da comunidade Front-End no mundo. Esse guia será atualizado ao decorrer das nossas experiências e tecnologias novas que surgirem.

O projeto está hospedado no Github. Para acessa-lo clique aqui. Você pode contribuir com o projeto enviando um e-mail para ronaldo@alfaiataria.com com seu Username no github. Após o envio clone o projeto da seguinte maneira: 

git clone https://github.com/Ronaldo-Maciel/guideline-front-alfa.git
Digite seu usuário e senha do github e já estará apto a contribuir.
